<https://webmaka.com/>

## リポジトリ

## サービス概要

- **サービス名**: うぇぶまか
- **コンセプト**: Crowdworksの手数料を回避し、直接依頼を受けるためのフリーランスWeb制作サービス

### ターゲット

- 個人事業主
- 中小企業
- スタートアップ

### 提供サービス

- コーポレートサイト制作
- ランディングページ制作
- 基本的なWeb制作全般

## サイト構成

| ページ | 内容 |
| --- | --- |
| トップページ | キャッチコピー、サービス説明、料金、実績プレビュー、問い合わせフォーム |
| ポートフォリオページ | 詳細な制作事例（後から追加可能） |

## デザインの方向性

**目標**: 信頼感 + 親しみやすさの両立

### 参考サイト

### 特徴

- 真っ白ではない温かみのある背景色
- 落ち着いた緑をメインに使用
- 柔らかく、相談しやすい雰囲気
- 「うぇぶまか」の名前と相性の良いナチュラルなトーン
